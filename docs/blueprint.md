# **App Name**: BookSwap Local

## Core Features:

- User Authentication: Email/password based authentication for readers and libraries.
- Role Management: Differentiate users from shop roles, managing different access and functionality.
- Book Listing: Users can add books available for swapping with title, author, ISBN (optional) and city.
- Book Search: List books available for swapping, filtered by city.
- Wishlist: Users can create a wishlist of books they want to read.
- Swap Matching: Matches books available to swap with books in wishlists.
- Swap Code Generation: Generate a unique swap code upon matching.
- Swap Check-in: Check-in swap at a partner library using the generated code.
- Swap Pass System: Implement a 'Swap Pass' system that allows 3 swaps/month, or unlimited swaps with payment.
- Library Dashboard: Display list of completed swaps and the generated income.

## Style Guidelines:

- Primary color: Deep orange (#D4741F) for a warm, inviting feel.
- Background color: Light gray (#EEEEEE) to provide a neutral and readable base.
- Accent color: Forest green (#4CAF50) for important actions, reinforcing concepts of sharing and community.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and friendly feel, ensuring readability across devices.
- Use line icons from the Material Design library, filled with #4CAF50.
- Mobile-first layout using a simple grid system to ensure responsiveness.
- Use subtle transitions and animations to enhance user experience.
